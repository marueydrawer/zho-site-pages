# zho-site-pages

## 项目简介

zho-site-pages 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名、网站或项目，仅作为 HTML 静态页面的存储与分发空间。  
所有页面内容均以独立、可访问的 HTML 文件形式存放，便于查看、测试或引用。

## 目录说明

```
zho-site-pages/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面依赖的静态资源（如 CSS、JS、图片等）
├── index.html      # 可选：页面索引或导航入口
└── README.md       # 本文件
```

- `pages/`：每个页面为一个独立子目录或单个 HTML 文件，命名清晰，便于区分。
- `assets/`：全局或页面共享的资源文件，按需组织。
- `index.html`：如使用，可作为浏览所有页面的起始页。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接在浏览器中打开。
- 页面之间相互独立，无内部依赖（除共享资源外）。
- 归档页面可能包含示例、测试、演示或存档内容，不保证长期可用性。
- 本仓库不提供页面内容的更新承诺，仅作存储用途。

## 维护说明

- 仓库维护者会不定期添加或清理页面，以保持内容整洁。
- 如需提交新页面，请遵循以下原则：
  - 页面文件放入 `pages/` 目录，命名无歧义。
  - 页面应自包含，不依赖外部不可控资源。
  - 避免在页面中包含任何敏感、侵权或违反平台政策的内容。
- 欢迎通过 Issue 或 Pull Request 提出建议或报告问题，但仓库不保证及时响应。
